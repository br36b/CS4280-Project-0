Program can be compiled using provided Makefile.

`./P0 [< filename] `
`./P0 file.fl2021 | file (.fl2021)` - implicit ending

Assumes that pipe < can be any format. Will allow any file for that operation. Anything else requires correct .fl2021 extension, implicit or explicit for anything else.

Sets are used to filter out duplicates.

Traversals are both printed to screen and output into files.
